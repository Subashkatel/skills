# Skill audit rubric

## Trigger quality
- Specific domain terms.
- Clear use cases.
- Avoids vague “helper” descriptions.

## Body quality
- Concise workflow.
- Evidence standard.
- Clear output shape.
- No stale model-specific overprompting.

## Domain quality
- Architecture: drivers, views, ADRs, tradeoffs.
- GPU: correctness, baseline, profiling, measurements.
- Quantum/QEC: assumptions, algebra, noise model, decoder, confidence.

## Safety and maintainability
- No unexpected tool escalation.
- No hidden-reasoning extraction instructions.
- Support files one level deep.
- Scripts are deterministic and documented.
