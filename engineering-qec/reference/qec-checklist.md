# QEC checklist

## Code definition
- Code family and parameters.
- Stabilizer/gauge generators.
- Boundaries and layout.
- Logical operators.
- Syndrome extraction schedule.

## Algebra checks
- Stabilizer commutation.
- Gauge/stabilizer relation for subsystem codes.
- Logical operators commute with checks.
- Logical X/Z anticommutation.
- Distance claim verified or labelled.

## Noise model
- Phenomenological or circuit-level.
- Gate, measurement, reset, idle, leakage, correlated, or biased noise.
- Pauli/stabilizer assumptions stated.

## Decoder
- Algorithm and library.
- Detector error model or parity-check input.
- Weight model.
- Boundary handling.
- Batch decoding and timing.

## Experiment
- Distances and rounds.
- Physical error rates.
- Shots and seeds.
- Confidence intervals.
- Logical observables.
- Result schema and plots.
